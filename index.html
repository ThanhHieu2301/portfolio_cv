MY SKILLS

HTML

HTML a markup language used to create the structure and format of web pages. It uses tags to mark and differentiate content within the document. Each tag has a name and is enclosed within angle brackets (< >).

CSS

CSS is a markup language used to format and style web pages. It is a fundamental and essential part of web development, allowing you to adjust the appearance of HTML elements and enhance the user experience.

JAVASCRIPT

JavaScript is a high-level, interpreted programming language commonly used for web development. It allows you to add dynamic and interactive elements to websites, enabling better user experiences. JavaScript is a versatile language that runs in the web browser, making it an essential part of front-end web development.

SASS

Sass is a preprocessor scripting language that enhances the capabilities of traditional CSS (Cascading Style Sheets). Sass allows developers to write CSS in a more organized and efficient manner by providing additional features and functionalities.

LARAVEL

Laravel is a popular open-source PHP web application framework that provides developers with a robust and elegant toolkit to build web applications and APIs. It follows the Model-View-Controller (MVC) architectural pattern, promoting clean and structured code.

SQL

SQL is a standard programming language used for managing and manipulating relational databases. SQL allows users to interact with databases to store, retrieve, update, and delete data. It serves as a bridge between humans and databases, making it easy to work with large amounts of information.
